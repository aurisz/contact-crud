# Contact CRUD

Simple React app built with hooks for CRUD operation

[LIVE DEMO](https://contact-crud.aurisophanz.now.sh/)

## Setup

```sh
git clone https://github.com/aurisz/contact-crud.git
```

```sh
cd contact-crud
```

```sh
npm install
```

```sh
npm start
```

visit `http://localhost:3000`

## Run Tests

```sh
npm test
```

## Run Production

```sh
npm run build
```

```sh
serve -s build
```

visit `http://localhost:5000`
